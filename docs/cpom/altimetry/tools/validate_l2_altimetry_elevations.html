cpom.altimetry.tools.validate_l2_altimetry_elevations

cpom.altimetry.tools.validate_l2_altimetry_elevations.py

This tool compares a selected month of radar altimetry mission elevation data against laser atimetry reference elevations. It also allows comparing a reference mission to itself.

Supported Missions

  • Altimetry Missions: CS2 (Native L2 and CryoTempo), S3A, S3B, ENVISAT, ERS1, ERS2
  • Reference Missions: ICESat-2 (ATL06), IceBridge (ILATM2, ILUTP2), Pre-IceBridge (BRMCR2, BLATM2), ICESat1 (GLAH12)

Please note you may need to add your dataset to 'get_default_variables' if you are testing with a none standard altimetry data product.

Notes to user

When comparing to icebridge : The default radius increases from 20m to 10000m, to capture enough points for validation. For the best performance alter the radius and max_diff argument for your usecase, or provide a DEM to correct reference locations to align with altimetry measurements.

When running for Cryotempo, the Cryotempo_Modes argument must be set.

Command Line Options

Required

  • reference_dir: Path to reference dataset.
  • altim_dir: Path to altimetry dataset.
  • year: Year of data.
  • month: Month of data.
  • area: CPOM area.
  • outdir: Output directory.

Optional

  • dem: Provide a DEM to correct location when comparing to IceBridge with a large radius, used in 'correct_elevation_using_slope'
  • radius: Search radius (in meters) for validation points.
  • max_diff: Maximum elevation difference between matched points.
  • add_vars: Additional variables to include in output (e.g., uncertainty).
  • cryotempo_modes: CryoTempo modes to include (required if using CryoTempo data).
  • max_workers: Number of parallel processing threads.
  • compare_to_historical_reference: (default: False) Set True when comparing to : IceBridge/PreIcebridge or ICESAT-1. Increases the max radius to 10000m. It is advised to use DEM location correction in this mode.
  • compare_to_self: Compare reference data points to nearby reference points (default: False).
  • bins: Number of bins for output data plot.

For a full list of the command line descriptions run: validate_l2_altimetry_elevations -h

Example Useage

Run for cs2 data sin and lrm mode, with 2 beams.  85def setup_logging(

196def get_variable(nc: Dataset, nc_var_path: str) -> np.ndarray:
197    """Retrieve variable from NetCDF file, handling groups if necessary.
198
199    Args:
200        nc (Dataset): The dataset object
201        nc_var_path (str): The path to the variable within the file,
202                        with groups separated by '/'.
203
204    Raises:
205        KeyError: If the variable or group is not found in the file.
206
207    Returns:
208        np.array: The retrieved variable as an array.
209    """
210    try:
211        parts = nc_var_path.split("/")
212        var = nc
213        for part in parts:
214            var = var[part]
215            if var is None:
216                raise IndexError(f"NetCDF parameter '{nc_var_path}' not found.")
217        return var[:]
218    except IndexError as err:
219        raise IndexError(f"NetCDF parameter or group {err} not found") from err

314def get_files_in_dir(
315    directory: Path,
316    year: str,
317    month: str,
318    filetype: str,
319) -> list[Path]:
320    """
321    Find .nc or.h5 files in the specified directory.
322
323    Args:
324        directory (Path): The directory to search for files.
325        year (str): Year to process
326        month (str): Month to process (0 filled)
327        filetype (str) : nc or h5
328    Returns:
329        List[Path]: A list of found .nc or .NC files with their full paths.
330    """
331    extensions = {"nc": "*.nc", "h5": "*.h5"}
332    all_files: list[Path] = []
333    pattern = extensions[filetype]
334    all_files.extend(Path(directory).rglob(pattern))
335
336    # If no files found, try upper case suffix
337    if not all_files:
338        all_files.extend(Path(directory).rglob(pattern.upper()))
339
340    # Matches: YYYYMMDDT | YYYYMMDD_ | YYMMDD_ |YYYYMMDDHHMMSS
341    file_basename_regex = re.compile(
342        rf"(?:{year}{month}\d{{2}}T\d*|{year}{month}\d{{2}}_\d*|{year[-2:]}{month}\d{{2}}_|{year}{month}\d{{8}})"  # noqa # pylint: disable=C0301
343    )
344
345    valid_files = [  # Exclude filenames containing multiple dots before the extension
346        f
347        for f in all_files
348        if any(
349            file_basename_regex.search(str(part)) for part in f.parts
350        )  # Check if regex matches any part of the path
351        and "." not in f.stem  # Exclude filenames containing multiple dots before the extension
352    ]
353
354    if valid_files:
355        return valid_files
356
357    # Match to YYYY.MM.DD in the input directory if no files found (Used for ISAT1 data load)
358    valid_files = [f for f in all_files if re.compile(rf"{year}\.{month}\.\d{{2}}").search(str(f))]
359    if valid_files:
360        return valid_files
361
362    return []

365class ProcessData:
366    """Class to process laser and altimetry data files by extracting and filtering
367    elevation data. To be run by a multiproccessor.
368
369    Methods:
370        get_altimetry_data_array: Extracts and processes altimetry data from NetCDF files.
371        get_is2_data_array : Extracts and processes is1 elevation data from HDF5 files.
372        get_is2_data_array : Extracts and processes is2 elevation data from HDF5 files.
373        get_icebridge_data_array : Extracts and processes icebridge data from NetCDF files.
374        fill_empty_latlon_with_nadir: Fills missing lat/lon values using nadir coordinates.
375        get_cryotempo_filters : Filters cryotempo data to valid modes.
376    """
377
378    def __init__(self, args, area, log):
379        self.args = args
380        self.area = area
381        self.log = log
382
383    def get_cryotempo_filters(self, nc, args):
384        """Check Cryotempo data is in a valid mode."""
385        if args.cryotempo_modes == "all":
386            return None
387
388        mode_map = {"lrm": 1, "sar": 2, "sin": 3}
389        valid_modes = {mode_map[mode] for mode in args.cryotempo_modes if mode in mode_map}
390
391        instrument_mode = get_variable(nc, "instrument_mode")
392        valid_mask = np.isin(instrument_mode, list(valid_modes))
393        return valid_mask if valid_mask.any() else None
394
395    def fill_empty_latlon_with_nadir(
396        self, nc: str, lat: np.ndarray, lon: np.ndarray, config: dict
397    ) -> tuple[np.ndarray, np.ndarray]:
398        """
399        Populate empty poca lat/lon with nadir lat/lon
400        Args:
401            nc (str): NetCDF filename
402            lat (np.array): Array of latitudes
403            lon (np.array): Array of longitudes
404            config (dict): Dictionary of variable names
405        Returns:
406            (np.array, np.arrray): Filled lat/lon arrays
407        """
408        bad_indices = np.isnan(lon)  # np.flatnonzero(lon.mask)  # Find empty longitude values
409        if bad_indices.size > 0:
410            lat_nadir = get_variable(nc, config["lat_nadir"])
411            lon_nadir = get_variable(nc, config["lon_nadir"])
412            lat[bad_indices] = lat_nadir[bad_indices]
413            lon[bad_indices] = np.mod(lon_nadir[bad_indices], 360)
414        return lat, lon

416    def get_altimetry_data_array(self, filename: Path) -> Optional[np.ndarray]:
417        """Extract and filter data from an altimetry data file.
418        Args:
419            filename (Path): Altimetry data filename (*.nc)
420        Raises:
421            ValueError: If variable lengths do not match.
422        Returns:
423            np.ndarray: Structured nd.array containing x,y,h fields
424        """
425        try:
426            with Dataset(filename) as nc:
427                config = get_default_variables(filename)
428                if config is None:
429                    self.log.error("Unsupported file basename %s for file", filename)
430                    return None
431
432                lats = get_variable(nc, config["lat"])
433                lons = np.mod(get_variable(nc, config["lon"]), 360)
434
435                if "lat_nadir" in config and "lon_nadir" in config:
436                    lats, lons = self.fill_empty_latlon_with_nadir(nc, lats, lons, config)
437
438                lats, lons, _, _ = self.area.inside_latlon_bounds(lats, lons)
439                x, y = self.area.latlon_to_xy(lats, lons)
440
441                if self.args.cryotempo_modes is not None:  # Filter modes for cryotempo
442                    surface_type_mask = self.get_cryotempo_filters(nc, self.args)
443                    if surface_type_mask is not None:
444                        x, y = x[surface_type_mask], y[surface_type_mask]
445                    else:
446                        self.log.info(
447                            "File does not contain %s data : %s",
448                            self.args.cryotempo_modes,
449                            filename,
450                        )
451
452                idx, _ = self.area.inside_mask(x, y)
453                if not idx.size:
454                    return None
455
456                try:
457                    x, y, elev = x[idx], y[idx], get_variable(nc, config["elev"])[idx]
458                except Exception as err:
459                    raise ValueError("Mismatch in variable lengths for x,y,h") from err
460
461                try:
462                    add_vars = self.args.add_vars or []
463                    additional_data = {
464                        var.rsplit("/", 1)[-1]: get_variable(nc, var)[idx]
465                        for var in add_vars
466                        if var in nc.variables
467                    }
468
469                    if list(additional_data.keys()) != add_vars:
470                        self.log.info(
471                            "Variable(s) %s missing from netcdf",
472                            set(add_vars) - set(additional_data.keys()),
473                        )
474                except Exception as err:
475                    raise ValueError("Mismatch in additional variable lengths.") from err
476
477            # Structured NumPy array containing x, y, elevation, and any additional variables
478            return np.array(
479                list(zip(x, y, elev, *(additional_data[var] for var in additional_data))),
480                dtype=[("x", "float64"), ("y", "float64"), ("h", "float64")]
481                + [(var, str(data.dtype)) for var, data in additional_data.items()],
482            )
483        except OSError as err:
484            self.log.error(f"Error loading altimetry data file {filename}, failed with : {err}")
485            return None

487    def get_is2_data_array(self, filename: Path) -> np.ndarray:
488        """Extract and filter data from an is2 data file.
489        Args:
490            filename (Path): IS2 data filename (.h5)
491        Raises:
492            ValueError: If variable lengths do not match
493        Returns:
494            np.ndarray: Structured nd.array containing x,y,h fields
495        """
496        try:
497            with h5py.File(filename, "r") as nc:
498                points = []
499                for beam in self.args.beams:
500                    config = {
501                        key: path.format(beam=beam)
502                        for key, path in get_default_variables(filename).items()
503                    }
504                    # Filter out files in wrong hemisphere. No files cross hemispheres
505                    try:
506                        hemisphere = get_variable(nc, config["lat"])[0]
507                        if (hemisphere < 0.0 and self.area.hemisphere == "north") or (
508                            hemisphere > 0.0 and self.area.hemisphere == "south"
509                        ):
510                            break
511
512                        elevation = get_variable(nc, config["elev"])
513                        ok = np.flatnonzero(
514                            (
515                                get_variable(nc, f"{beam}/land_ice_segments/atl06_quality_summary")
516                                == 0
517                            )
518                            & (elevation <= 10e3)
519                        )
520                        if len(ok) == 0:
521                            continue
522

587    def get_is1_data_array(self, filename: Path) -> Optional[np.ndarray]:  # pylint: disable=R0914
588        """Extract and filter IS1 data from a file.
589        Follows method from Smith et al., Science(2020)
590        Args:
591            filename (Path): IS1 data filename
592        Raises:
593            ValueError: If variable lengths do not match
594        Returns:
595            np.ndarray: Structured nd.array containing x,y,h fields
596        """
597
598        def _get_crs_transformers():
599            """Get the CRS and transformers for converting coordinates from TOPEX to WGS84."""
600            # Convert geodetic coordinates to cartesian coordinates
601            topex_crs = CRS.from_proj4("+proj=latlong +a=6378136.300 +rf=298.257 +no_defs")
602            ecef_crs = CRS.from_proj4("+proj=geocent  +a=6378136.300 +rf=298.257 +no_defs")
603            return (
604                Transformer.from_crs(topex_crs, ecef_crs, always_xy=True),
605                Transformer.from_crs(ecef_crs, CRS.from_epsg(4979), always_xy=True),
606            )
607
608        with h5py.File(filename, "r") as file:
609            config = get_default_variables(filename)
610            # Get the hemisphere of the file
611            hemisphere = get_variable(file, config["lat"])[0]
612            if (hemisphere < 0.0 and self.area.hemisphere == "north") or (
613                hemisphere > 0.0 and self.area.hemisphere == "south"
614            ):
615                return None
616
617            elevation_tpx = get_variable(file, config["elev"])
618            valid_mask = (
619                (file["Data_40HZ"]["Waveform"]["i_numPk"][:] == 1)  # N. peaks in returned echo
620                & (file["Data_40HZ"]["Quality"]["elev_use_flg"][:] == 0)  # Flag to use elevation
621                & (file["Data_40HZ"]["Quality"]["sigma_att_flg"][:] == 0)  # Attitude quality flag
622                & (file["Data_40HZ"]["Quality"]["sat_corr_flg"][:] < 3)  # Saturation corr flag
623                & (elevation_tpx != file[config["elev"]].attrs["_FillValue"])
624            )
625
626            if not valid_mask.any():
627                self.log.error("No valid data found in file %s", filename)
628                return None
629
630            lats_tpx, lons_tpx, elevation_tpx = (
631                get_variable(file, config["lat"])[valid_mask],
632                get_variable(file, config["lon"])[valid_mask],
633                elevation_tpx[valid_mask],
634            )
635            sat_corr = file["Data_40HZ"]["Elevation_Corrections"]["d_satElevCorr"][:][valid_mask]
636
637            topex_to_ecef, ecef_to_wgs = _get_crs_transformers()
638            # Convert geodetic coordinates to cartesian coordinates
639            x_ecef, y_ecef, h_ecef = topex_to_ecef.transform(lons_tpx, lats_tpx, elevation_tpx)
640            # Convert tpx cartesian coordinates to wgs84
641            lons, lats, elevation = ecef_to_wgs.transform(x_ecef, y_ecef, h_ecef)
642            # Transform lat, lon to x,y in appropriate polar stereo projections
643            lons = np.mod(lons, 360)
644            lats, lons, bounded_indices, _ = self.area.inside_latlon_bounds(lats, lons)
645            x, y = self.area.latlon_to_xy(lats, lons)
646
647            # Apply saturation correction
648            elevation = elevation[bounded_indices] + sat_corr[bounded_indices]
649
650            if not len(x) == len(y) == len(elevation):
651                raise ValueError("Mismatch in variable lengths for x,y,h")
652
653            return np.array(
654                list(zip(x, y, elevation)),
655                dtype=[("x", "float64"), ("y", "float64"), ("h", "float64")],
656            )

675def get_elev_differences(
676    args: argparse.Namespace,
677    laser_points: np.ndarray,
678    altimeter_points: np.ndarray,
679    prefix: str = "",
680    nearest_only=False,
681) -> dict:
682    """
683    Calculate the elevation differences between IS2 points and altimeter points
684    within a specified search radius and maximum elevation difference.
685    Args:
686        args (argparse.Namespace): Configuration parameters
687        laser_points (np.ndarray): Is2 array containing x,y,h
688        altimeter_points (np.ndarray): Altimetry array containing x,y,h
689        prefix(str): Reference dataset prefix.
690        nearest_only (Boolean): Compare altimetry to only the nearest reference point
691    Returns:
692        dict: Dh between altimetry and is2 elevations.
693    """
694
695    is2_tree = KDTree(np.c_[laser_points["x"], laser_points["y"]])
696    add_vars = [var for var in altimeter_points.dtype.names if var not in {"x", "y", "h"}]
697    results: dict = {
698        key: []
699        for key in [
700            "dh",
701            "sep_dist",
702            f"{prefix}x",
703            f"{prefix}y",
704            f"{prefix}h",
705            "reference_x",
706            "reference_y",
707            "reference_h",
708        ]
709        + add_vars
710    }
711
712    for altimeter_point in altimeter_points:
713        if nearest_only:
714            dist, idx = is2_tree.query((altimeter_point["x"], altimeter_point["y"]))
715            indices = [idx] if dist <= args.radius else []
716        else:
717            indices = is2_tree.query_ball_point(  # Find reference points within search_radius
718                (altimeter_point["x"], altimeter_point["y"]), args.radius
719            )
720
721        if args.compare_to_self:  # Remove comparision to self
722            indices = [
723                i
724                for i in indices
725                if (laser_points[i]["x"], laser_points[i]["y"])
726                != (altimeter_point["x"], altimeter_point["y"])
727            ]
728
729        for idx in indices:
730            ref_point = laser_points[idx]
731            dh = altimeter_point["h"] - ref_point["h"]
732
733            if np.abs(dh) <= args.maxdiff:
734                results["dh"].append(dh)
735                results["sep_dist"].append(
736                    np.sqrt(
737                        (altimeter_point["x"] - ref_point["x"]) ** 2
738                        + (altimeter_point["y"] - ref_point["y"]) ** 2
739                    )
740                )
741                results[f"{prefix}x"].append(altimeter_point["x"])
742                results[f"{prefix}y"].append(altimeter_point["y"])
743                results[f"{prefix}h"].append(altimeter_point["h"])
744                results["reference_x"].append(ref_point["x"])
745                results["reference_y"].append(ref_point["y"])
746                results["reference_h"].append(ref_point["h"])
747
748                for var in add_vars:
749                    results[var].append(altimeter_point[var])
750
751    return results

754def correct_elevation_using_slope(data, args, log, prefix) -> dict:
755    """Correct for differences in the measurement location
756    of the reference and altimetry data point.
757    Conceptually shift reference location, to be located at altimetry datapoint,
758    using DEM slope.
759    Args:
760        data (nd.array): Output of "get_elev_differences"
761        args (argparse.Namespace): Configuration parameters
762        log (Logger): Logger
763        prefix (str): Reference data prefix
764    Returns
765       dict: Slope corrected Dh between altimetry and is2 elevations.
766    """
767    log.info("Performing slope correction")
768    dem = Dem(args.dem)
769    dh_dem_elev = (dem.interp_dem(data[f"{prefix}x"], data[f"{prefix}y"])) - (
770        dem.interp_dem(data["reference_x"], data["reference_y"])
771    )
772    data["reference_h"] = data["reference_h"] + dh_dem_elev
773    data["dh"] = data["dh"] - dh_dem_elev
774
775    data = {key: np.array(data[key]) for key in data}
776    valid_mask = (~np.isnan(data["dh"])) & (np.abs(data["dh"]) <= args.maxdiff)
777
778    return {key: val[valid_mask] for key, val in data.items()}
