cpom.altimetry.geolocation.geolocate_roemer

Slope correction/geolocation function using an adapted Roemer method from : Roemer, S., Legrésy, B., Horwath, M., and Dietrich, R.: Refined analysis of radar altimetry data applied to the region of the subglacial Lake Vostok/Antarctica, Remote Sens. Environ., 106, 269–284, https://doi.org/10.1016/j.rse.2006.02.026, 2007.

Adaption by A.Muir, M.McMillan, Q.Huang (CPOM), includes a two stage POCA location, performed by minimizing the range to the satellite from DEM points located within firstly a circular area around the nadir location of radius equal to half the beam width, and then within the pulse limited footprint around the initial POCA at a finer resampled DEM resolution.

Next the calculated POCA's range to satellite is checked to see if it is within the range window (configurably trimmed). If not the slope correction is failed.

Finally the slope correction and height is calculated from

height[i] = altitudes[i] - (geo_corrected_tracker_range[i] + retracker_correction[i]) + slope_correction[i]

 42def calculate_distances3d(
 43    x1_coord: float,
 44    y1_coord: float,
 45    z1_coord: float,
 46    x2_array: np.ndarray | list,
 47    y2_array: np.ndarray | list,
 48    z2_array: np.ndarray | list,
 49    squared_only=False,
 50) -> list[float]:
 51    """calculates the distances between a  refernce cartesian point (x1,y1,z1) in 3d space
 52    and a list of other points : x2[],y2[],z2[]
 53
 54    Args:
 55        x1_coord (float): x coordinate of ref point
 56        y1_coord (float): y coordinate of ref point
 57        z1_coord (float): z coordinate of ref point
 58        x2_array (np.ndarray): list of x coordinates
 59        y2_array (np.ndarray): list of y coordinates
 60        z2_array (np.ndarray): list of z coordinates
 61        squared_only (bool) : if True, only calculate the squares of diffs and not sqrt
 62                              this will be faster, but doesn't give actual distances
 63
 64    Returns:
 65        list[float]: list of distances between points x1,y1,z1 and x2[],y2[],z2[]
 66    """
 67
 68    x2_array = np.array(x2_array)
 69    y2_array = np.array(y2_array)
 70    z2_array = np.array(z2_array)
 71
 72    distances = (x2_array - x1_coord) ** 2 + (y2_array - y1_coord) ** 2 + (z2_array - z1_coord) ** 2
 73
 74    if not squared_only:
 75        distances = np.sqrt(distances)
 76
 77    return distances  # Convert back to a regular Python list

 80def calculate_distances2d(
 81    x1_coord: float,
 82    y1_coord: float,
 83    x2_array: np.ndarray,
 84    y2_array: np.ndarray,
 85    squared_only=False,
 86) -> np.ndarray:
 87    """calculates the distances between a  refernce cartesian point (x1,y1) in 2d space
 88    and a list of other points : x2[],y2[]
 89
 90    Args:
 91        x1_coord (float): x coordinate of ref point
 92        y1_coord (float): y coordinate of ref point
 93        x2_array (np.ndarray): list of x coordinates
 94        y2_array (np.ndarray): list of y coordinates
 95        squared_only (bool) : if True, only calculate the squares of diffs and not sqrt
 96                              this will be faster, but doesn't give actual distances
 97
 98    Returns:
 99        np.ndarray: list of distances between points x1,y1,z1 and x2[],y2[]
100    """
101
102    x2_array = np.array(x2_array)
103    y2_array = np.array(y2_array)
104
105    distances = (x2_array - x1_coord) ** 2 + (y2_array - y1_coord) ** 2
106
107    if squared_only:
108        distances = (x2_array - x1_coord) ** 2 + (y2_array - y1_coord) ** 2
109    else:
110        distances = np.sqrt((x2_array - x1_coord) ** 2 + (y2_array - y1_coord) ** 2)
111
112    return distances

115def find_poca(
116    zdem: np.ndarray,
117    xdem: np.ndarray,
118    ydem: np.ndarray,
119    nadir_x: float,
120    nadir_y: float,
121    alt_pt: float,
122):
123    """Function that finds the POCA using method similar to Roemer et al. (2007)
124       Finds the point with the shortest range to the satellite in the DEM segment and
125       computes the slope correction to height
126       Adapted from original : CLS (python) of McMillan (Matlab) code
127
128    Args:
129        zdem (np.ndarray): DEM height values
130        xdem (np.ndarray): x locations of DEM in polar stereo coordinates (m)
131        ydem (np.ndarray): y locations of DEM in polar stereo coordinates (m)
132        nadir_x (float): x location of nadir in polar stereo coordinates (m)
133        nadir_y (float): y location of nadir in polar stereo coordinates (m)
134        alt_pt (float): altitude at nadir (m)
135
136    Returns:
137        (float,float,float,float,float,bool): poca_x, poca_y, poca_z, slope_correction_to_height,
138        range_to_satellite_of_poca,
139        flg_success
140    """
141
142    # ----------------------------------------------------------------
143    # compute horizontal plane distance to each cell in beam footprint
144    # ----------------------------------------------------------------
145
146    # compute x and y distance of all points in beam footprint from nadir coordinate of
147    # current record
148    dem_dx_vec = xdem - nadir_x
149    dem_dy_vec = ydem - nadir_y
150
151    # Compute the squared magnitude of distance from nadir
152    dem_dmag_squared = dem_dx_vec**2 + dem_dy_vec**2
153
154    # Account for earth curvature using the squared distance
155    dem_dz_vec = zdem - alt_pt - dem_dmag_squared / (2.0 * EARTH_RADIUS)
156
157    dem_range_vec = np.sqrt(dem_dmag_squared + (dem_dz_vec) ** 2)
158
159    # find range to, and indices of, closest dem pixel
160    [dem_rpoca, dempoca_ind] = np.nanmin(dem_range_vec), np.nanargmin(dem_range_vec)
161
162    if np.isnan(zdem[dempoca_ind]) | (zdem[dempoca_ind] == -9999):
163        return -999, -999, -999, -999, -999, 0
164
165    # compute relocation correction to apply to assumed nadir altimeter elevation to move to poca
166    slope_correction_to_height = dem_rpoca + zdem[dempoca_ind] - alt_pt
167
168    flg_success = 1
169
170    return (
171        xdem[dempoca_ind],
172        ydem[dempoca_ind],
173        zdem[dempoca_ind],
174        slope_correction_to_height,
175        dem_rpoca,
176        flg_success,
177    )

180def datetime2year(date_dt):
181    """calculate decimal year from datetime
182
183    Args:
184        date_dt (datetime): datetime obj to process
185
186    Returns:
187        float: decimal year
188    """
189    year_part = date_dt - datetime(year=date_dt.year, month=1, day=1)
190    year_length = datetime(year=date_dt.year + 1, month=1, day=1) - datetime(
191        year=date_dt.year, month=1, day=1
192    )
193    return date_dt.year + year_part / year_length

196def geolocate_roemer(
197    lats: np.ndarray,
198    lons: np.ndarray,
199    altitudes: np.ndarray,
200    thisdem: Dem | None,
201    thisdem_fine: Dem | None,
202    config: dict,
203    surface_type_20_ku: np.ndarray,
204    geo_corrected_tracker_range: np.ndarray,
205    retracker_correction: np.ndarray,
206    points_to_include: np.ndarray,
207) -> Tuple[np.ndarray, np.ndarray, np.ndarray, np.ndarray, np.ndarray]:
208    """Geolocate CS2 LRM measurements using an adapted Roemer (Roemer et al, 2007) method
209
210    Args:
211        l1b (Dataset): NetCDF Dataset of L1b file
212        thisdem (Dem): Dem object used for Roemer correction
213        thisdem_fine (Dem): Dem object used for fine Roemer correction (maybe same obj as thisdem)
214        config (dict): config dictionary containing
215            "roemer_geolocation": {
216                "fine_grid_sampling": 10,
217                "max_poca_reloc_distance": 6600,
218                "range_window_lower_trim": 0,
219                "range_window_upper_trim": 0,
220                "median_filter": False,
221                "median_filter_width": 7,
222                "reject_outside_range_window": True,
223                "use_sliding_window": False,
224            }
225            "instrument": {
226                "across_track_beam_width": 15000, # meters
227                "pulse_limited_footprint_size": 1600, # meters
228            }
229        surface_type_20_ku (np.ndarray): surface type for track, where 1 == grounded_ice
230        geo_corrected_tracker_range (np.ndarray) : geo-corrected tracker range (NOT retracked)
231        retracker_correction (np.ndarray) : retracker correction to range (m)
232        points_to_include (np.ndarray) : boolean array of points to include (False == reject)
233    Returns:
234        (np.ndarray, np.ndarray, np.ndarray, np.ndarray, np.ndarray):
235        (height_20_ku, lat_poca_20_ku, lon_poca_20_ku, slope_ok, relocation_distance)
236    """
237
238    if thisdem is None:
239        raise ValueError("thisdem None value passed")
240    if thisdem_fine is None:
241        raise ValueError("thisdem_fine None value passed")
242
243    # ------------------------------------------------------------------------------------
244    # Retrieve configuration parameters (to avoid repeatedly getting from config dict
245    # during loops further down the function)
246    # ------------------------------------------------------------------------------------
247
248    across_track_beam_width = config["instrument"]["across_track_beam_width"]  # meters
249    pulse_limited_footprint_size = config["instrument"]["pulse_limited_footprint_size"]  # m
250    reference_bin_index = config["instrument"]["ref_bin_index"]
251    range_bin_size = config["instrument"]["range_bin_size"]  # meters
252    num_bins = config["instrument"]["num_range_bins"]
253
254    # Additional options
255
256    max_poca_reloc_distance = config["roemer_geolocation"]["max_poca_reloc_distance"]
257
258    fine_grid_sampling = config["roemer_geolocation"]["fine_grid_sampling"]
259
260    median_filter_dem_segment = config["roemer_geolocation"]["median_filter"]
261    # Adjusted to be close to CS2 PLF width of 1600m.
262    median_filter_width = config["roemer_geolocation"]["median_filter_width"]
263    reject_outside_range_window = config["roemer_geolocation"]["reject_outside_range_window"]
264    range_window_lower_trim = config["roemer_geolocation"]["range_window_lower_trim"]
265    range_window_upper_trim = config["roemer_geolocation"]["range_window_upper_trim"]
266
267    # ------------------------------------------------------------------------------------
268
269    # Check that longitude is in range 0..360
270    lons = lons % 360.0
271
272    # Transform to X,Y locs in DEM projection
273    nadir_x, nadir_y = thisdem.lonlat_to_xy_transformer.transform(
274        lons, lats
275    )  # pylint: disable=unpacking-non-sequence
276
277    # Create working parameter arrays
278    poca_x = np.full_like(nadir_x, dtype=float, fill_value=np.nan)
279    poca_y = np.full_like(nadir_x, dtype=float, fill_value=np.nan)
280    poca_z = np.full_like(nadir_x, dtype=float, fill_value=np.nan)
281    slope_correction = np.full_like(nadir_x, dtype=float, fill_value=np.nan)
282    slope_ok = np.full_like(nadir_x, dtype=bool, fill_value=True)
283    height_20_ku = np.full_like(nadir_x, dtype=float, fill_value=np.nan)
284    relocation_distance = np.full_like(nadir_x, dtype=float, fill_value=np.nan)
285
286    # ------------------------------------------------------------------------------------
287    #  Loop through each track record
288    # ------------------------------------------------------------------------------------
289
290    for i, _ in enumerate(nadir_x):
291        # By default, set POCA x,y to nadir, and height to Nan
292        poca_x[i] = nadir_x[i]
293        poca_y[i] = nadir_y[i]
294        poca_z[i] = np.nan
295
296        # if record is excluded due to previous checks, then skip
297        if not points_to_include[i]:
298            continue
299
300        # get the rectangular bounds about the track, adjusted for across track beam width and
301        # the dem posting
302        x_min = nadir_x[i] - (across_track_beam_width / 2 + thisdem.binsize)
303        x_max = nadir_x[i] + (across_track_beam_width / 2 + thisdem.binsize)
304        y_min = nadir_y[i] - (across_track_beam_width / 2 + thisdem.binsize)
305        y_max = nadir_y[i] + (across_track_beam_width / 2 + thisdem.binsize)
306
307        segment = [(x_min, x_max), (y_min, y_max)]
308
309        # Extract the rectangular segment from the DEM
310        try:
311            xdem, ydem, zdem = thisdem.get_segment(segment, grid_xy=True, flatten=False)
312        except (IndexError, ValueError, TypeError, AttributeError, MemoryError):
313            slope_ok[i] = False
314            continue
315        except Exception:  # pylint: disable=W0718
316            slope_ok[i] = False
317            continue
318
319        if median_filter_dem_segment:
320            smoothed_zdem = median_filter(zdem, size=median_filter_width)
321            zdem = smoothed_zdem
322
323        # Step 1: find the DEM points within a circular area centred on the nadir
324        # point corresponding to a radius of half the beam width
325        xdem = xdem.flatten()
326        ydem = ydem.flatten()
327        zdem = zdem.flatten()
328
329        # Compute 2d distance between each dem location and nadir in (x,y)
330        dem_to_nadir_dists = calculate_distances2d(nadir_x[i], nadir_y[i], xdem, ydem)
331
332        # find where dem_to_nadir_dists is within beam. ie extract circular area
333        include_dem_indices = np.where(
334            np.array(dem_to_nadir_dists) < (across_track_beam_width / 2.0)
335        )[0]
336        if len(include_dem_indices) == 0:
337            slope_ok[i] = False
338            continue
339
340        xdem = xdem[include_dem_indices]
341        ydem = ydem[include_dem_indices]
342        zdem = zdem[include_dem_indices]
343
344        # Check remaining DEM points for bad height values and remove
345        nan_mask = np.isnan(zdem)
346        include_only_good_zdem_indices = np.where(~nan_mask)[0]
347        if len(include_only_good_zdem_indices) < 1:
348            slope_ok[i] = False
349            continue
350
351        xdem = xdem[include_only_good_zdem_indices]
352        ydem = ydem[include_only_good_zdem_indices]
353        zdem = zdem[include_only_good_zdem_indices]
354
355        # Only keep DEM heights which are in a sensible range
356        # this step removes DEM values set to most fill_values
357        valid_dem_heights = np.where(np.abs(zdem) < 5000.0)[0]
358        if len(valid_dem_heights) < 1:
359            slope_ok[i] = False
360            continue
361
362        xdem = xdem[valid_dem_heights]
363        ydem = ydem[valid_dem_heights]
364        zdem = zdem[valid_dem_heights]
365
366        # Find the POCA location and slope correction to height
367        (
368            this_poca_x,
369            this_poca_y,
370            this_poca_z,
371            slope_correction_to_height,
372            range_to_sat_of_poca,
373            flg_success,
374        ) = find_poca(zdem, xdem, ydem, nadir_x[i], nadir_y[i], altitudes[i])
375
376        if not flg_success:
377            slope_ok[i] = False
378            continue
379        poca_x[i] = this_poca_x
380        poca_y[i] = this_poca_y
381        poca_z[i] = this_poca_z
382
383        if fine_grid_sampling > 0:
384            # Create finer grid resolution around poca
385            # -------------------------------------
386
387            # get the rectangular bounds around the approx POCA,
388            # adjusted for pulse limited width and the dem posting
389
390            x_min = poca_x[i] - (pulse_limited_footprint_size / 2 + thisdem.binsize)
391            x_max = poca_x[i] + (pulse_limited_footprint_size / 2 + thisdem.binsize)
392            y_min = poca_y[i] - (pulse_limited_footprint_size / 2 + thisdem.binsize)
393            y_max = poca_y[i] + (pulse_limited_footprint_size / 2 + thisdem.binsize)
394
395            segment = [(x_min, x_max), (y_min, y_max)]
396
397            # Extract the rectangular segment from the DEM
398            try:
399                xdem, ydem, zdem = thisdem_fine.get_segment(segment, grid_xy=False, flatten=False)
400            except (IndexError, ValueError, TypeError, AttributeError, MemoryError):
401                slope_ok[i] = False
402                continue
403            except Exception:  # pylint: disable=W0718
404                slope_ok[i] = False
405                continue
406
407            # Define new grid for finer resolution
408            grid_x, grid_y = np.mgrid[
409                x_min:x_max:fine_grid_sampling, y_min:y_max:fine_grid_sampling
410            ]
411            grid_x = grid_x.flatten()
412            grid_y = grid_y.flatten()
413
414            new_z = interpn(
415                (np.flip(ydem.copy()), xdem),
416                np.flip(zdem.copy(), 0),
417                (grid_y, grid_x),
418                method="linear",
419                bounds_error=False,
420                fill_value=np.nan,
421            )
422
423            # Compute distance between dem points and the approximate POCA in (x,y,z)
424            dem_to_poca_dists = calculate_distances2d(poca_x[i], poca_y[i], grid_x, grid_y)
425
426            # find where dem_to_poca_dists is within PLF. ie extract circular area
427            include_dem_indices = np.where(
428                np.array(dem_to_poca_dists) < (pulse_limited_footprint_size / 2.0)
429            )[0]
430            if len(include_dem_indices) == 0:
431                slope_ok[i] = False
432                continue
433
434            xdem = grid_x[include_dem_indices]
435            ydem = grid_y[include_dem_indices]
436            zdem = new_z[include_dem_indices]
437
438            # Check remaining DEM points for bad height values and remove
439            nan_mask = np.isnan(zdem)
440            include_only_good_zdem_indices = np.where(~nan_mask)[0]
441            if len(include_only_good_zdem_indices) < 1:
442                slope_ok[i] = False
443                continue
444
445            xdem = xdem[include_only_good_zdem_indices]
446            ydem = ydem[include_only_good_zdem_indices]
447            zdem = zdem[include_only_good_zdem_indices]
448
449            # Only keep DEM heights which are in a sensible range
450            # this step removes DEM values set to most fill_values
451            valid_dem_heights = np.where(np.abs(zdem) < 5000.0)[0]
452            if len(valid_dem_heights) < 1:
453                slope_ok[i] = False
454                continue
455
456            xdem = xdem[valid_dem_heights]
457            ydem = ydem[valid_dem_heights]
458            zdem = zdem[valid_dem_heights]
459
460            # Find the POCA location and slope correction to height
461            (
462                this_poca_x,
463                this_poca_y,
464                this_poca_z,
465                slope_correction_to_height,
466                range_to_sat_of_poca,
467                flg_success,
468            ) = find_poca(zdem, xdem, ydem, nadir_x[i], nadir_y[i], altitudes[i])
469
470            if not flg_success:
471                slope_ok[i] = False
472                continue
473
474            poca_x[i] = this_poca_x
475            poca_y[i] = this_poca_y
476            poca_z[i] = this_poca_z
477
478            if reject_outside_range_window:
479                range_to_window_start = (
480                    range_window_lower_trim
481                    + geo_corrected_tracker_range[i]
482                    - (reference_bin_index) * range_bin_size
483                )
484                range_to_window_end = (
485                    geo_corrected_tracker_range[i]
486                    + ((num_bins - reference_bin_index) * range_bin_size)
487                    - range_window_upper_trim
488                )
489                if (range_to_sat_of_poca < range_to_window_start) or (
490                    range_to_sat_of_poca > range_to_window_end
491                ):
492                    slope_ok[i] = False
493                    continue
494
495            slope_correction[i] = slope_correction_to_height
496            relocation_distance[i] = np.sqrt(
497                (this_poca_x - nadir_x[i]) ** 2 + (this_poca_y - nadir_y[i]) ** 2
498            )
499            if relocation_distance[i] > max_poca_reloc_distance:
500                slope_ok[i] = False
501
502        if config["roemer_geolocation"]["use_sliding_window"]:
503            # Step 1: Calculate all distances once
504            all_distances_flat = calculate_distances3d(
505                x1_coord=nadir_x[i],
506                y1_coord=nadir_y[i],
507                z1_coord=altitudes[i],
508                x2_array=xdem.flatten(),
509                y2_array=ydem.flatten(),
510                z2_array=zdem.flatten(),
511                squared_only=False,
512            )
513
514            all_distances = np.array(all_distances_flat).reshape(xdem.shape)
515
516            # Step 2: Slide the window over the pre-calculated distance grid
517            min_distance = np.inf
518            min_position = (0, 0)
519            window_size = (int)(pulse_limited_footprint_size / thisdem.binsize)
520
521            for ii in range(all_distances.shape[0] - window_size + 1):
522                for jj in range(all_distances.shape[1] - window_size + 1):
523                    # Extract the current window of distances
524                    window = all_distances[ii : ii + window_size, jj : jj + window_size]
525
526                    # Calculate the mean distance within the window
527                    mean_distance = np.mean(window)
528
529                    # Update the minimum mean and position if a new minimum is found
530                    if mean_distance < min_distance:
531                        min_distance = mean_distance
532                        min_position = (ii, jj)
533
534            # min_position is the position of the window with the smallest mean distance
535
536            # --------------------------------------------------------------------------------------
537            #  Find Location of POCA x,y
538            # --------------------------------------------------------------------------------------
539
540            poca_x[i] = xdem[min_position]
541            poca_y[i] = ydem[min_position]
542
543            # --------------------------------------------------------------------------------------
544            #  Find Location of POCA z
545            # --------------------------------------------------------------------------------------
546
547            poca_z[i] = zdem[min_position]
548
549            # --------------------------------------------------------------------------------------
550            #  Calculate Slope Correction
551            # --------------------------------------------------------------------------------------
552
553            dem_to_sat_dists = calculate_distances3d(
554                nadir_x[i],
555                nadir_y[i],
556                altitudes[i],
557                [poca_x[i]],
558                [poca_y[i]],
559                [poca_z[i]],
560            )

500            if relocation_distance[i] > max_poca_reloc_distance:
501                slope_ok[i] = False
502
503        if config["roemer_geolocation"]["use_sliding_window"]:
504            # Step 1: Calculate all distances once
505            all_distances_flat = calculate_distances3d(
506                x1_coord=nadir_x[i],
507                y1_coord=nadir_y[i],
508                z1_coord=altitudes[i],
509                x2_array=xdem.flatten(),
510                y2_array=ydem.flatten(),
511                z2_array=zdem.flatten(),
512                squared_only=False,
513            )
514
515            all_distances = np.array(all_distances_flat).reshape(xdem.shape)
516
517            # Step 2: Slide the window over the pre-calculated distance grid
518            min_distance = np.inf
519            min_position = (0, 0)
520            window_size = (int)(pulse_limited_footprint_size / thisdem.binsize)
521
522            for ii in range(all_distances.shape[0] - window_size + 1):
523                for jj in range(all_distances.shape[1] - window_size + 1):
524                    # Extract the current window of distances
525                    window = all_distances[ii : ii + window_size, jj : jj + window_size]
526
527                    # Calculate the mean distance within the window
528                    mean_distance = np.mean(window)
529
530                    # Update the minimum mean and position if a new minimum is found
531                    if mean_distance < min_distance:
532                        min_distance = mean_distance
533                        min_position = (ii, jj)
534
535            # min_position is the position of the window with the smallest mean distance
536
537            # --------------------------------------------------------------------------------------
538            #  Find Location of POCA x,y
539            # --------------------------------------------------------------------------------------
540
541            poca_x[i] = xdem[min_position]
542            poca_y[i] = ydem[min_position]
543
544            # --------------------------------------------------------------------------------------
545            #  Find Location of POCA z
546            # --------------------------------------------------------------------------------------
547
548            poca_z[i] = zdem[min_position]
549
550            # --------------------------------------------------------------------------------------
551            #  Calculate Slope Correction
552            # --------------------------------------------------------------------------------------
553
554            dem_to_sat_dists = calculate_distances3d(
555                nadir_x[i],
556                nadir_y[i],
557                altitudes[i],
558                [poca_x[i]],
559                [poca_y[i]],
560                [poca_z[i]],
561            )
562
563            # Calculate the slope correction to height
564            slope_correction[i] = dem_to_sat_dists[0] + poca_z[i] - altitudes[i]
565
566    # Transform all POCA x,y to lon,lat
567    lon_poca_20_ku, lat_poca_20_ku = thisdem.xy_to_lonlat_transformer.transform(poca_x, poca_y)
568
569    # Calculate height as altitude-(corrected range)+slope_correction
570    height_20_ku = np.full_like(lats, np.nan)
571
572    for i in range(len(lats)):  # pylint: disable=consider-using-enumerate
573        if np.isfinite(geo_corrected_tracker_range[i]):
574            if slope_ok[i] and surface_type_20_ku[i] == 1:  # grounded ice type only
575                height_20_ku[i] = (
576                    altitudes[i]
577                    - (geo_corrected_tracker_range[i] + retracker_correction[i])
578                    + slope_correction[i]
579                )
580            else:
581                height_20_ku[i] = np.nan
582        else:
583            height_20_ku[i] = np.nan
584
585        # Set POCA lat,lon to nadir if no slope correction
586        if (
587            (not np.isfinite(lat_poca_20_ku[i]))
588            or (not np.isfinite(lon_poca_20_ku[i]))
589            or (not slope_ok[i])
590        ):
591            lat_poca_20_ku[i] = lats[i]
592            lon_poca_20_ku[i] = lons[i]
593            height_20_ku[i] = np.nan
594
595    return (height_20_ku, lat_poca_20_ku, lon_poca_20_ku, slope_ok, relocation_distance)

Geolocate CS2 LRM measurements using an adapted Roemer (Roemer et al, 2007) method
